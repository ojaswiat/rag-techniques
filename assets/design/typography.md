# Typography — COMP702 Proposal / Dissertation / Slides

Single shared scale across Word documents and PowerPoint, per `palette.md`'s
"Oxford Ink" identity. Based on a standard 1.25 modular scale, rounded to
clean point sizes so variation between levels stays moderate.

## Fonts

| Role | Font |
|---|---|
| Body text (paragraphs, lists) | Optima |
| Headings (H1–H4) | Optima |
| Diagrams, charts, tables, captions, labels | Calibri |
| Code / monospace | Consolas |

## Paragraph rules

- Body paragraphs: justified alignment, 1.15 line spacing, 8pt space-after,
  first-line indent none (block style with spacing instead).
- Headings: 12pt space-before, 6pt space-after, keep-with-next enabled.
- No em dashes anywhere; use commas, semicolons, or plain hyphens for ranges.
- British English spelling throughout.
